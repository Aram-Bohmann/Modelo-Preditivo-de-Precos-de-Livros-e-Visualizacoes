# Modelo-Preditivo-de-Precos-de-Livros-e-Visualizacoes
### Descrição
Este repositório armazena os códigos de um modelo preditivo de preços de livros e visualizações acerca da tabela sobre livros, criado com Python e desenvolvida para complementar o TCC do técnico em ciência de dados.

#### Descrição do Modelo
O modelo (app.py) foi implementado em Python e utiliza a biblioteca Scikit-Learn para construção do algoritmo preditivo. A aplicação final foi disponibilizada em uma interface interativa construída com Streamlit, permitindo ao usuário visualizar dados, acompanhar o processo de treinamento do modelo e testar previsões diretamente pela interface.
